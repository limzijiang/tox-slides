## 📦 內容

- 五大 Toxidrome（交感／抗膽鹼／膽鹼性／鴉片／鎮靜安眠）
- 病史不可靠的三大陷阱與問家屬技巧
- VS / PE / ECG 暗號（含 Na 通道阻斷四指紋）
- 尿液 tox 完整假陽性／假陰性清單
- 解毒劑速查 + 活性碳 + MDAC + WBI 適應症
- 四大致命併發症處置（抽搐／躁動／高體溫／心律不整）
